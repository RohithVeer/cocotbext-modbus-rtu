# cocotbext-modbus

A reusable Cocotb VIP for Modbus RTU protocol verification.

## Features

- Modbus RTU Driver and Monitor
- CRC error checking
- Scoreboard for result comparison
- Function code coverage
- HTML/Text reporting

git clone https://github.com/yourname/cocotbext-modbus
